This is a fun, interactive guessing game built with HTML and Tailwind CSS.These guessing game contains 3 modes Number,Places and Animal.The website gives hint for the places and Animal and notification of "Too high" or "too low" for the number part.Users have freedom to change between the light and dark mode and change the theme to blue,green or to purple.After the correct answer the question refreshes.These works on both phone and pc.The website gives you visual response (SHRINKS SLIGHTLY) as you press the button.It contains centred layout as it looks attractive
